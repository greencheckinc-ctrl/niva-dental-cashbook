📄 Step-by-step setup
1. Go to console.firebase.google.com → Create project
2. Build → Realtime Database → Create database
3. Authentication → Sign-in method → Enable Email/Password
4. Authentication → Users → Add user (clinic email + password)
5. Realtime Database → Rules → paste: {"rules":{"$synckey":{"$uid":{".read":"auth!=null && auth.uid==$uid",".write":"auth!=null && auth.uid==$uid"}}}}
6. Project Settings → Your apps → Add Web app → Copy config
7. Paste config + email + password above → Save & Connect
